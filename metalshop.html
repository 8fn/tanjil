004 : MetalShop E-Commerce

MetalShop is a web app e-commerce platform built with the MERN stack, utilizing MongoDB, Express, React, and Node.js for a dynamic and efficient user experience. Check my project here

This web application is a full-featured e-commerce platform built with the MERN stack, employing a client-server architecture for scalability and maintainability. The server side is powered by the Express framework, while the frontend is developed using React with Redux for robust state management. The platform offers a dual-purpose functionality: customers can schedule home visits for technicians to perform repairs and can also shop for household appliances, placing orders seamlessly. Users can track their order history and manage their purchases, with payments processed securely through PayPal. For administrators, the application includes comprehensive management tools to oversee appliances, users, and orders, as well as handle scheduling and repair requests, ensuring efficient operations and customer satisfaction. This integration of shopping and service scheduling makes the application a versatile solution for household needs.
